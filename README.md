#youtube-altyazi-sniffer-js

Bu Chrome eklentisi, YouTube videolarındaki otomatik altyazıları yakalayıp
popup arayüzünde veya YouTube sayfasının altında görüntülemenizi sağlar.

### Özellikler
- YouTube altyazılarını otomatik algılar
- JSON3 ve XML altyazı formatlarını çözer
- Tüm altyazıyı tek pencerede gösterir
- Kopyalama butonu ile metni kolayca dışa aktarabilirsiniz
- YouTube dışında çalışmaz

### Kurulum
1. Bu projeyi indirin.
2. Chrome → Extensions → Developer Mode açın.
3. “Load unpacked” deyip klasörü seçin.

### Not
Bu proje tamamen JavaScript + HTML tabanlıdır.
